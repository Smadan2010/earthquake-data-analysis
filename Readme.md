# Earthquake Data Analysis

This is an end-to-end earthquake data analysis project built using SQL, Python, and Streamlit.

## Project Overview
The project analyzes global earthquake data to identify seismic trends such as:
- Strongest earthquakes
- Deep vs shallow earthquakes
- Tsunami-related events
- Regional earthquake activity

An interactive Streamlit dashboard is used to visualize insights.

## Tools Used
- Python
- MySQL
- SQLAlchemy
- Pandas
- Streamlit

## How to Run
1. Install required packages  
   pip install -r requirements.txt

2. Run the Streamlit app  
   streamlit run test.py
